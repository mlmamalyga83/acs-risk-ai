# ACS ECG Detector

Детекция ЭКГ-признаков Острого Коронарного Синдрома с интерпретацией.

## Быстрый старт

```bash
# Установка
pip install -r requirements.txt
python test_imports.py

# Проверка системы
python run.py --stage check
python run.py --stage init

# Скачать базы и запустить обучение
./scripts/download_all.sh
python run.py --stage all --tune

# Запустить интерфейс
streamlit run src/app/main.py
# → http://localhost:8501
```

## Что делает модель

Принимает 12-канальную ЭКГ + клинические данные (возраст, пол, тропонин, АД).
Выдаёт риск-скор ОКС 0-100% + Grad-CAM тепловую карту + автоматическое заключение.

⚠️ Исследовательский прототип. Не медицинское изделие.

## Данные

- PTB-XL: 21,837 записей (Германия) — обучение
- MIT-BIH ST-T: 28 записей (США) — внешняя валидация
